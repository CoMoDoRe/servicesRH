# GROUP
%SYNTAX_GROUP%
Sintaxis: 12GROUP {JOIN|LEAVE|LIST} [nick] [contraseña]
%%

%HELP_GROUP%
El agrupamiento de nicks trata de poder tener todos tus nicks agrupados compartiendo
todas las propiedades con tu nick principal (contraseña, email, ip virtual, etc).

Para agrupar un nick, necesitas estar utilizando el nick que quieres agrupar con tu
12nick principal y utilizar tu 12contraseña para agruparlo.

Una vez hayas realizado el agrupamiento, podrás utilizar tu segundo nick como si fuera
el principal.

Las distintas opciones del comando 12GROUP son:

12- JOIN: Si estás utilizando un nick sin registrar, lo agrupas a tu nick principal.
Los parámetros 12nick y 12contraseña son requeridos para esta acción.
12- LEAVE: Si estás utilizando un nick agrupado, eliminará a este del grupo quedando
desregistrado. No requiere parámetros.
12- LIST: Lista los nicks dentro de tu grupo de nicks. No requiere parámetros.

Ejemplo: Si tu nick principal es Venganito y su contraseña es 12345, para agrupar tu nick
con el nick Venganito harías: 12/msg %N GROUP JOIN Venganito 12345
%%

%GROUP_JOIN_ACCOUNT_EXISTS%
04Error: Este nick ya está registrado o agrupado.
%%

%GROUP_JOIN_PRIMARY_DOESNT_EXIST%

Tu nick ahora forma parte del grupo de 12%s.
